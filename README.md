# job4j_dreamJob

Technologies:
- Servlets
- JSP
- Simple HTML/JS(JQueue|Ajax)/CSS
- JDBC
- Apache Tomcat
- OOP
- MVC Pattern
- Mockito
- slf4j/log4j
- PostgreSQL
- Travis CI / CodeCov / CheckStyle

This application demonstrates a simple job placement and resume service for job search and candidate search
